# 资讯分享网站

## 技术栈
后端使用Spring Boot，前端使用React搭建。

## 项目地址
[后端](https://github.com/zijian-z/news)
[前端](https://github.com/zijian-z/news-react)

## 功能
- [x] 登录注册
- [x] 分享资讯
- [x] 发表评论
- [x] 用户通知
- [x] 个人信息维护
- [x] 热度排序
