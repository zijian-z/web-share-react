# 模仿Hacker News编写的链接分享网站

## 技术栈
前端使用React，后端使用Spring Boot搭建

## 项目地址
[前端](https://github.com/zijian-z/web-share-react)
[后端](https://github.com/zijian-z/web-share)

## 功能
- [x] 登录注册
- [x] 分享链接
- [x] 发表评论
- [x] @用户通知
- [x] 个人信息维护
- [x] 热度排序
